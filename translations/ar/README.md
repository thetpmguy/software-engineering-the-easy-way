البرمجيات فعلًا، وستكتب برامجك الأولى، وستنشر موقعًا إلكترونيًا حقيقيًا، وستتمكن من مجاراة
أي حوار تقني بثقة — كل ذلك مشروح بلغة يومية بسيطة، وبتشبيهات من الحياة العادية.

## 🤔 لمن هذه الدورة؟

- «أستخدم التطبيقات كل يوم لكن **ليست لدي أدنى فكرة عن كيفية عمل أي شيء منها فعلًا**.»
- «أعمل *مع* مهندسين (مديرًا، مصمِّمة، مؤسِّسًا، محامية، طبيبًا…) وقد سئمت من
  **هز رأسي موافقًا في الاجتماعات**.»
- «أريد **الانتقال مهنيًا إلى مجال التقنية**، لكن كل دورة تفترض أنني أعرف أشياء لا أعرفها.»
- «لديّ **فضول** لأعرف كيف بُني هذا العالم الرقمي من حولي.»

لا توجد **أي متطلبات مسبقة** — نشرح كل شيء، حتى ما هو «الملف» (file) حقًا.

## 🎓 ماذا ستتعلم

- كيف تعمل الحواسيب والبرمجيات والإنترنت وقواعد البيانات (databases) والسحابة (cloud)
  والذكاء الاصطناعي (AI) فعلًا.
- كيف تقرأ وتكتب برامج بسيطة بلغة Python.
- كيف تستخدم Git و GitHub كالمحترفين (ستنشر موقعك الإلكتروني الخاص على الإنترنت).
- كيف تخطط فرق البرمجيات الحقيقية منتجاتها وتبنيها وتختبرها وتطلقها.
- كيف تتحدث مع المهندسين ندًا لند — واجهات البرمجة (APIs) والخوادم (servers) والأخطاء
  البرمجية (bugs) والنشر (deploys) والسبرنتات (sprints)، كلها مفكوكة الشيفرة.

**ما لن نغطيه:** الرياضيات المتقدمة، أو أدوات الشركات المتخصصة، أو الضجيج الإعلامي. كل تشبيه
نستخدمه نخبرك أيضًا أين يتوقف عن الصحة، ووحدة الذكاء الاصطناعي صادقة بشأن حدوده.
