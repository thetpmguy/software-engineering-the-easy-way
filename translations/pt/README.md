Se você consegue ler esta página, você consegue terminar este curso. Ao final, você vai entender
como o software realmente funciona, escrever seus primeiros programas, publicar um site de
verdade e participar de qualquer conversa técnica de igual para igual — tudo explicado em
linguagem simples, com analogias do dia a dia.

## 🤔 Para quem é isto?

- "Eu uso aplicativos todos os dias, mas **não faço a menor ideia de como tudo isso realmente funciona**."
- "Eu trabalho *com* engenheiros (gerente, designer, fundador, advogado, médico…) e estou cansado de
  **ficar só balançando a cabeça nas reuniões**."
- "Quero **mudar de carreira para a área de tecnologia**, mas todo curso pressupõe coisas que eu não sei."
- "Tenho **curiosidade** de saber como o mundo digital ao meu redor é construído."

**Não há nenhum pré-requisito** — explicamos tudo, inclusive o que um "arquivo (file)" realmente é.

## 🎓 O que você vai aprender

- Como computadores, software, internet, bancos de dados (databases), a nuvem (cloud) e a IA realmente funcionam.
- Como ler e escrever programas simples em Python.
- Como usar Git e GitHub como um profissional (você vai publicar seu próprio site no ar).
- Como equipes de software de verdade planejam, constroem, testam e lançam produtos.
- Como conversar com engenheiros de igual para igual — APIs, servidores (servers), bugs, deploys e sprints, tudo decifrado.

**O que não vamos cobrir:** matemática avançada, ferramentas corporativas específicas ou
modismos vazios. Toda analogia também avisa onde ela deixa de funcionar, e o módulo de IA é
honesto sobre os limites.

## 🚀 Como Começar

1. **⭐ Dê uma estrela (star) neste repositório** para encontrar o caminho de volta depois.
2. Leia o **[START HERE](../../START_HERE.md)** (5 minutos) — como o curso funciona e as
   ferramentas gratuitas de navegador (browser) que você vai usar. **Não é preciso instalar nada.**
   Todos os laboratórios rodam no navegador, até no celular.
3. Comece pelo [Módulo 01](../../modules/01-what-is-a-computer/README.md) — ou escolha um atalho
   nas **[Trilhas de Aprendizado](../../LEARNING_PATHS.md)** ("Só tenho um fim de semana", "Eu
   gerencio engenheiros", "Só me interesso por IA"…).
4. Travou em alguma palavra? O **[Glossário](../../GLOSSARY.md)** define cada termo em uma única frase simples.

> **Professores e grupos de estudo:** tudo é livre para reutilizar sob a [licença](../../LICENSE).
> Faça um fork do repositório e adapte-o para a sua sala de aula.

## 📝 Quizzes & Laboratórios

- Os **quizzes** ficam dentro da pasta de cada módulo (`quiz.md`) — as respostas vêm escondidas
  sob dobras "Show answer", e todo quiz termina com um desafio de *"explique para um amigo"*.

  gratuitas de navegador, que funcionam até no celular. É fazendo os laboratórios que o aprendizado gruda.
