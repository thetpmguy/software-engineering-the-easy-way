Si puedes leer esta página, puedes terminar este curso. Al final entenderás cómo funciona
realmente el software, escribirás tus primeros programas, publicarás un sitio web de verdad
y podrás participar con seguridad en cualquier conversación técnica — todo explicado en
lenguaje cotidiano, con analogías de la vida diaria.

## 🤔 ¿Para quién es este curso?

- "Uso aplicaciones todos los días, pero **no tengo ni idea de cómo funcionan por dentro**."
- "Trabajo *con* ingenieros (como gerente, diseñadora, fundador, abogada, médico…) y estoy harto de
  **asentir con la cabeza en las reuniones**."
- "Quiero **cambiar de carrera hacia la tecnología**, pero todos los cursos dan por sabidas cosas que no sé."
- "Tengo **curiosidad** por saber cómo está construido el mundo digital que me rodea."

No hay **ningún requisito previo** — te explicamos todo, incluso qué es realmente un "archivo" (file).

## 🎓 Qué vas a aprender

- Cómo funcionan de verdad las computadoras, el software, internet, las bases de datos (databases),
  la nube (cloud) y la inteligencia artificial (AI).
- Cómo leer y escribir programas sencillos en Python.
- Cómo usar Git y GitHub como un profesional (publicarás tu propio sitio web en vivo).
- Cómo los equipos de software reales planifican, construyen, prueban y lanzan productos.
- Cómo hablar con ingenieros de igual a igual — APIs, servidores (servers), errores (bugs),
  despliegues (deploys) y sprints, todos descifrados.

**Lo que no cubriremos:** matemáticas avanzadas, herramientas empresariales específicas ni puro humo.
Cada analogía te dice también dónde deja de funcionar, y el módulo de IA es honesto sobre sus límites.

## 🚀 Cómo empezar

1. **⭐ Dale una estrella a este repositorio** para que siempre encuentres el camino de vuelta.
2. Lee **[EMPIEZA AQUÍ](../../START_HERE.md)** (5 minutos) — cómo funciona el curso y las
   herramientas gratuitas de navegador que usarás. **No hay que instalar nada.** Cada laboratorio
   funciona en un navegador web, incluso en un teléfono.
3. Comienza con el [Módulo 01](../../modules/01-what-is-a-computer/README.md) — o elige un atajo en
   las **[Rutas de aprendizaje](../../LEARNING_PATHS.md)** ("Tengo un fin de semana", "Dirijo un
   equipo de ingenieros", "Solo me interesa la IA"…).
4. ¿Te atascaste con una palabra? El **[Glosario](../../GLOSSARY.md)** define cada término en una
   sola frase sencilla.

> **Profesores y grupos de estudio:** todo se puede reutilizar gratis bajo la [licencia](../../LICENSE).
> Haz un fork del repositorio y adáptalo a tu aula.

## 📝 Cuestionarios y laboratorios

- Los **cuestionarios** viven dentro de la carpeta de cada módulo (`quiz.md`) — las respuestas están
  incluidas bajo pestañas desplegables de "Mostrar respuesta", y cada cuestionario termina con el
  reto de *"explícaselo a un amigo"*.
- Los **laboratorios** (`lab.md`) son prácticos y requieren **cero instalación** — solo herramientas
  gratuitas basadas en el navegador, aptas para el teléfono. Hacer los laboratorios es lo que hace
  que el aprendizaje se quede contigo.
