మీరు ఈ పేజీని చదవగలిగితే, ఈ కోర్సును పూర్తి చేయగలరు. కోర్సు ముగిసే సమయానికి
సాఫ్ట్‌వేర్ (software) నిజంగా ఎలా పనిచేస్తుందో అర్థం చేసుకుంటారు, మీ మొదటి
ప్రోగ్రామ్‌లు (programs) రాస్తారు, నిజమైన వెబ్‌సైట్ (website) ప్రచురిస్తారు, ఏ సాంకేతిక
సంభాషణలోనైనా ధైర్యంగా పాల్గొనగలుగుతారు — ఇవన్నీ రోజువారీ జీవితం నుంచి తీసుకున్న
పోలికలతో, పూర్తిగా సులభమైన భాషలో వివరించబడ్డాయి.

## 🤔 ఇది ఎవరి కోసం?

- "నేను రోజూ యాప్‌లు (apps) వాడతాను, కానీ **అవి నిజంగా ఎలా పనిచేస్తాయో నాకు అస్సలు తెలియదు**."
- "నేను ఇంజినీర్లతో *కలిసి* పనిచేస్తాను (మేనేజర్, డిజైనర్, వ్యవస్థాపకుడు, లాయర్, డాక్టర్…)
  — **మీటింగ్‌లలో అర్థం కాకుండానే తల ఊపడం** నాకు విసుగొచ్చింది."
- "నేను **టెక్ రంగంలోకి కెరీర్ మారాలనుకుంటున్నాను**, కానీ ప్రతి కోర్సూ నాకు తెలియని
  విషయాలు ముందే తెలుసని భావిస్తుంది."
- "నా చుట్టూ ఉన్న డిజిటల్ ప్రపంచం ఎలా నిర్మించబడిందో తెలుసుకోవాలనే **కుతూహలం** నాకుంది."

**ముందస్తు అర్హతలు ఏవీ అవసరం లేదు** — "ఫైల్" (file) అంటే నిజంగా ఏమిటో సహా అన్నింటినీ మేమే వివరిస్తాం.

## 🎓 మీరు నేర్చుకునేది

- కంప్యూటర్లు, సాఫ్ట్‌వేర్ (software), ఇంటర్నెట్ (internet), డేటాబేస్‌లు (databases),
  క్లౌడ్ (cloud), కృత్రిమ మేధ (AI) నిజంగా ఎలా పనిచేస్తాయో.
- Python భాషలో సరళమైన ప్రోగ్రామ్‌లను చదవడం, రాయడం ఎలాగో.
- Git, GitHub లను ఒక ప్రొఫెషనల్‌లా ఎలా వాడాలో (మీ సొంత లైవ్ వెబ్‌సైట్‌ను మీరే ప్రచురిస్తారు).
- నిజమైన సాఫ్ట్‌వేర్ బృందాలు ఉత్పత్తులను ఎలా ప్రణాళిక చేసి, నిర్మించి, పరీక్షించి,
  విడుదల చేస్తాయో.
- ఇంజినీర్లతో సమానంగా ఎలా మాట్లాడాలో — API లు, సర్వర్లు (servers), బగ్‌లు (bugs),
  డిప్లాయ్‌లు (deploys), స్ప్రింట్‌లు (sprints) — అన్నీ విడమరిచి చెప్పబడ్డాయి.

**మేము బోధించనివి:** ఉన్నత స్థాయి గణితం, నిర్దిష్ట ఎంటర్‌ప్రైజ్ సాధనాలు, లేదా అతిశయోక్తులు.
ప్రతి పోలికా అది ఎక్కడ సరిపోదో కూడా చెబుతుంది; AI మాడ్యూల్ దాని పరిమితుల గురించి
నిజాయితీగా మాట్లాడుతుంది.

## 🚀 ప్రారంభించడం ఎలా

1. **⭐ ఈ రిపోకు (repo) స్టార్ (star) ఇవ్వండి** — తిరిగి రావడానికి దారి సులభంగా దొరుకుతుంది.
2. **[ఇక్కడ ప్రారంభించండి](../../START_HERE.md)** చదవండి (5 నిమిషాలు) — కోర్సు ఎలా
   పనిచేస్తుందో, మీరు వాడబోయే ఉచిత బ్రౌజర్ (browser) సాధనాలు ఏమిటో తెలుస్తుంది.
   **ఏదీ ఇన్‌స్టాల్ (install) చేయాల్సిన అవసరం లేదు.** ప్రతి ల్యాబ్ (lab) వెబ్
   బ్రౌజర్‌లోనే నడుస్తుంది — ఫోన్‌లో కూడా.
3. [మాడ్యూల్ 01](../../modules/01-what-is-a-computer/README.md) తో మొదలుపెట్టండి — లేదా
   **[లెర్నింగ్ పాత్‌ల](../../LEARNING_PATHS.md)** నుంచి ఒక షార్ట్‌కట్ ఎంచుకోండి
   ("నాకు ఒక వీకెండ్ మాత్రమే ఉంది", "నేను ఇంజినీర్లను మేనేజ్ చేస్తాను",
   "నాకు AI మాత్రమే కావాలి"…).
4. ఏదైనా పదం దగ్గర ఆగిపోయారా? **[పదకోశం](../../GLOSSARY.md)** ప్రతి పదాన్నీ ఒకే ఒక్క
   సులభమైన వాక్యంలో వివరిస్తుంది.

> **ఉపాధ్యాయులు & స్టడీ గ్రూపులు:** [లైసెన్స్](../../LICENSE) కింద అన్నీ ఉచితంగా తిరిగి
> వాడుకోవచ్చు. రిపోను ఫోర్క్ (fork) చేసి మీ తరగతి గదికి తగ్గట్టుగా మార్చుకోండి.
