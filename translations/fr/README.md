Si vous pouvez lire cette page, vous pouvez terminer ce cours. À la fin, vous comprendrez comment
les logiciels fonctionnent vraiment, vous écrirez vos premiers programmes, vous publierez un vrai
site web et vous tiendrez votre place dans n'importe quelle conversation technique — le tout
expliqué en langage courant, avec des analogies tirées de la vie de tous les jours.

## 🤔 À qui s'adresse ce cours ?

- « J'utilise des applications tous les jours, mais **je n'ai aucune idée de comment tout cela
  fonctionne réellement**. »
- « Je travaille *avec* des ingénieurs (manager, designer, fondatrice, avocat, médecin…) et j'en ai
  assez de **hocher la tête en réunion**. »
- « Je veux **me reconvertir dans la tech**, mais chaque cours suppose des connaissances que je n'ai pas. »
- « Je suis simplement **curieux** de savoir comment le monde numérique qui m'entoure est construit. »

Il n'y a **aucun prérequis** — nous expliquons tout, y compris ce qu'est vraiment un « fichier » (file).

## 🎓 Ce que vous allez apprendre

- Comment fonctionnent vraiment les ordinateurs, les logiciels, internet, les bases de données
  (databases), le cloud et l'IA (AI).
- Comment lire et écrire des programmes simples en Python.
- Comment utiliser Git et GitHub comme un professionnel (vous publierez votre propre site web en ligne).
- Comment les vraies équipes logicielles planifient, construisent, testent et livrent leurs produits.
- Comment parler aux ingénieurs d'égal à égal — API, serveurs (servers), bugs, déploiements
  (deploys) et sprints, enfin décodés.

**Ce que nous ne couvrirons pas :** les mathématiques avancées, les outils d'entreprise spécifiques,
ou le battage médiatique. Chaque analogie vous dit aussi où elle atteint ses limites, et le module
sur l'IA est honnête sur ce qu'elle ne sait pas faire.

## 🚀 Pour commencer

1. **⭐ Mettez une étoile à ce dépôt** pour pouvoir retrouver votre chemin.
2. Lisez **[COMMENCEZ ICI](../../START_HERE.md)** (5 minutes) — comment le cours fonctionne et les
   outils gratuits dans le navigateur que vous utiliserez. **Rien à installer.** Chaque atelier
   fonctionne dans un navigateur web, même sur un téléphone.
3. Commencez par le [Module 01](../../modules/01-what-is-a-computer/README.md) — ou choisissez un
   raccourci dans les **[Parcours d'apprentissage](../../LEARNING_PATHS.md)** (« J'ai un week-end »,
   « Je manage des ingénieurs », « Seule l'IA m'intéresse »…).
4. Bloqué sur un mot ? Le **[Glossaire](../../GLOSSARY.md)** définit chaque terme en une seule
   phrase simple.

> **Enseignants et groupes d'étude :** tout est librement réutilisable sous la [licence](../../LICENSE).
> Forkez le dépôt et adaptez-le à votre classe.

## 📝 Quiz et ateliers

- Les **quiz** se trouvent dans le dossier de chaque module (`quiz.md`) — les réponses sont incluses
  sous des volets « Afficher la réponse », et chaque quiz se termine par un défi
  *« expliquez-le à un ami »*.
- Les **ateliers** (`lab.md`) sont pratiques et ne demandent **aucune installation** — uniquement
  des outils gratuits dans le navigateur, utilisables même sur téléphone. C'est en faisant les
  ateliers que l'apprentissage s'ancre vraiment.
