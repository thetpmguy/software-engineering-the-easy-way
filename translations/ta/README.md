இந்தப் பக்கத்தை உங்களால் படிக்க முடிந்தால், இந்தப் பாடநெறியை உங்களால் முடிக்கவும் முடியும்.
பாடநெறியின் இறுதியில், மென்பொருள் (software) உண்மையில் எப்படி வேலை செய்கிறது என்பதைப்
புரிந்துகொள்வீர்கள்; உங்கள் முதல் நிரல்களை (programs) எழுதுவீர்கள்; உண்மையான ஒரு
இணையதளத்தை (website) வெளியிடுவீர்கள்; எந்தத் தொழில்நுட்ப உரையாடலிலும் தன்னம்பிக்கையுடன்
பங்கேற்பீர்கள் — இவை அனைத்தும் அன்றாட வாழ்க்கையிலிருந்து எடுத்த உவமைகளுடன், முற்றிலும்
எளிய மொழியில் விளக்கப்பட்டுள்ளன.

## 🤔 இது யாருக்கானது?

- "நான் தினமும் ஆப்களைப் (apps) பயன்படுத்துகிறேன், ஆனால் **அவை உண்மையில் எப்படி வேலை
  செய்கின்றன என்பது எனக்குச் சுத்தமாகத் தெரியாது**."
- "நான் பொறியாளர்களுடன் *இணைந்து* வேலை செய்கிறேன் (மேலாளர், வடிவமைப்பாளர், நிறுவனர்,
  வழக்கறிஞர், மருத்துவர்…) — **கூட்டங்களில் புரியாமலே தலையாட்டுவது** எனக்குச் சலித்துவிட்டது."
- "நான் **தொழில்நுட்பத் துறைக்கு மாற** விரும்புகிறேன், ஆனால் ஒவ்வொரு பாடநெறியும் எனக்குத்
  தெரியாத விஷயங்களை ஏற்கனவே தெரியும் என்று எடுத்துக்கொள்கிறது."
- "என்னைச் சுற்றியுள்ள டிஜிட்டல் உலகம் எப்படிக் கட்டமைக்கப்பட்டுள்ளது என்று அறிய
  எனக்கு **ஆர்வம்**."

**முன்தேவைகள் எதுவும் இல்லை** — "கோப்பு" (file) என்றால் உண்மையில் என்ன என்பது உட்பட
எல்லாவற்றையும் நாங்கள் விளக்குகிறோம்.

## 🎓 நீங்கள் கற்றுக்கொள்ளப் போவது

- கணினிகள், மென்பொருள் (software), இணையம் (internet), தரவுத்தளங்கள் (databases),
  கிளவுட் (cloud), செயற்கை நுண்ணறிவு (AI) ஆகியவை உண்மையில் எப்படி வேலை செய்கின்றன.
- Python மொழியில் எளிய நிரல்களை எப்படிப் படிப்பது, எழுதுவது.
- Git மற்றும் GitHub-ஐ ஒரு நிபுணரைப் போலப் பயன்படுத்துவது எப்படி (உங்கள் சொந்த
  இணையதளத்தை நேரடியாக வெளியிடுவீர்கள்).
- உண்மையான மென்பொருள் குழுக்கள் எப்படித் திட்டமிட்டு, உருவாக்கி, சோதித்து, தயாரிப்புகளை
  வெளியிடுகின்றன.
- பொறியாளர்களுடன் சமமாக உரையாடுவது எப்படி — API, சர்வர்கள் (servers), பிழைகள் (bugs),
  டிப்ளாய்கள் (deploys), ஸ்பிரிண்ட்கள் (sprints) — எல்லாம் விளக்கப்பட்டுள்ளன.

**நாங்கள் கற்பிக்காதவை:** உயர்நிலைக் கணிதம், குறிப்பிட்ட நிறுவனக் கருவிகள், அல்லது வெற்று
விளம்பர வார்த்தைகள். ஒவ்வொரு உவமையும் அது எங்கே பொருந்தாது என்பதையும் சொல்கிறது; AI
பற்றிய பாடப்பகுதி அதன் வரம்புகளைப் பற்றி நேர்மையாகப் பேசுகிறது.

## 🚀 தொடங்குவது எப்படி

1. **⭐ இந்த ரெப்போவுக்கு (repo) நட்சத்திரம் (star) கொடுங்கள்** — திரும்பி வரும் வழியை
   எளிதாகக் கண்டுபிடிக்கலாம்.
2. **[இங்கே தொடங்குங்கள்](../../START_HERE.md)** என்பதைப் படியுங்கள் (5 நிமிடங்கள்) —
   பாடநெறி எப்படி வேலை செய்கிறது, நீங்கள் பயன்படுத்தப்போகும் இலவச உலாவி (browser)
   கருவிகள் என்ன என்பதை அறியலாம். **எதையும் நிறுவத் (install) தேவையில்லை.** ஒவ்வொரு
   பயிற்சிக்கூடமும் (lab) வெப் உலாவியிலேயே இயங்கும் — மொபைல் போனிலும் கூட.
3. [பாடப்பகுதி 01](../../modules/01-what-is-a-computer/README.md)-இல் தொடங்குங்கள் —
   அல்லது **[கற்றல் பாதைகளில்](../../LEARNING_PATHS.md)** இருந்து ஒரு குறுக்குவழியைத்
   தேர்ந்தெடுங்கள் ("என்னிடம் ஒரு வார இறுதிதான் உள்ளது", "நான் பொறியாளர்களை
   நிர்வகிக்கிறேன்", "எனக்கு AI மட்டும் போதும்"…).
4. ஏதாவது ஒரு சொல்லில் மாட்டிக்கொண்டீர்களா? **[சொற்களஞ்சியம்](../../GLOSSARY.md)**
   ஒவ்வொரு சொல்லையும் ஒரே ஒரு எளிய வாக்கியத்தில் விளக்குகிறது.

> **ஆசிரியர்கள் மற்றும் கற்றல் குழுக்களே:** [உரிமத்தின்](../../LICENSE) கீழ் அனைத்தையும்
> இலவசமாக மீண்டும் பயன்படுத்தலாம். ரெப்போவை ஃபோர்க் (fork) செய்து உங்கள் வகுப்பறைக்கு
> ஏற்றவாறு மாற்றிக்கொள்ளுங்கள்.

## 📝 வினாடி வினாக்களும் பயிற்சிக்கூடங்களும்

- **வினாடி வினாக்கள்** ஒவ்வொரு பாடப்பகுதியின் கோப்புறையிலும் (`quiz.md`) உள்ளன —
  விடைகள் "விடையைக் காட்டு" என்ற மடிப்புகளுக்குள் உள்ளன; ஒவ்வொரு வினாடி வினாவும்
  *"ஒரு நண்பருக்கு விளக்குங்கள்"* என்ற சவாலுடன் முடிகிறது.
- **பயிற்சிக்கூடங்கள்** (`lab.md`) நேரடிச் செயல்முறை சார்ந்தவை; **எந்த நிறுவலும்
  (installation) தேவையில்லை** — இலவச, உலாவி அடிப்படையிலான கருவிகள் மட்டுமே;
  மொபைலிலும் வேலை செய்யும். பயிற்சிக்கூடங்களைச் செய்யும்போதுதான் கற்றது மனதில் நிலைக்கும்.
