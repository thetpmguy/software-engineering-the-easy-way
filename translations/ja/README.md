このページが読めるなら、あなたはこのコースを最後までやり遂げられます。修了する頃には、
ソフトウェアが本当はどう動いているのかを理解し、初めてのプログラム（program）を書き、
本物のウェブサイトを公開し、どんな技術的な会話にも自信を持って加われるようになります —
すべて、身近な暮らしのたとえ話を交えた、やさしい言葉だけで説明されています。

## 🤔 これは誰のためのコース？

- 「毎日アプリを使っているけれど、**中身がどう動いているのか、まったく見当もつかない**。」
- 「エンジニアと*一緒に*働いている（マネージャー、デザイナー、創業者、弁護士、医師…）けれど、
  **会議でうなずいてごまかすのは**もううんざり。」
- 「**テック業界へのキャリアチェンジ**をしたいけれど、どのコースも私の知らないことを前提にしている。」
- 「身のまわりのデジタル世界がどう作られているのか、純粋に**知りたい**。」

**前提知識は一切不要です** — 「ファイル（file）」とは本当は何なのか、というところから、すべてを説明します。

## 🎓 このコースで学べること

- コンピューター、ソフトウェア、インターネット、データベース（database）、クラウド（cloud）、AI が実際にどう動くのか。
- Python でかんたんなプログラムを読み書きする方法。
- Git と GitHub をプロのように使う方法（自分のウェブサイトを実際に公開します）。
- 本物のソフトウェアチームがどうやって製品を企画し、作り、テストし、世に送り出すのか。
- エンジニアと対等に話す方法 — API、サーバー（server）、バグ（bug）、デプロイ（deploy）、スプリント（sprint）、ぜんぶ意味が分かるようになります。

**扱わないもの:** 高度な数学、特定の企業向けツール、そして誇大広告。どのたとえ話にも
「どこから先は当てはまらないか」を添えてあり、AI のモジュール（module）はその限界について正直に語ります。

## 📝 クイズとラボ

- **クイズ**は各モジュールのフォルダー内（`quiz.md`）にあります — 答えは「Show answer」の
  折りたたみの中に用意されていて、どのクイズも最後は*「友だちに説明してみよう」*という課題で締めくくられます。
- **ラボ**（`lab.md`）は手を動かす実習で、**インストールは一切不要** — 無料のブラウザー
  ベースのツールだけを使い、スマホでも取り組めます。学びが本当に身につくのは、ラボをやったときです。
