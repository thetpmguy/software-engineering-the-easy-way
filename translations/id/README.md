Kalau kamu bisa membaca halaman ini, kamu bisa menyelesaikan kursus ini. Di akhir kursus, kamu
akan paham bagaimana perangkat lunak (software) benar-benar bekerja, menulis program pertamamu,
menerbitkan situs web sungguhan, dan bisa ikut nimbrung dengan percaya diri dalam percakapan
teknis apa pun — semuanya dijelaskan dengan bahasa sehari-hari, lengkap dengan analogi dari
kehidupan kita.

## 🤔 Untuk siapa kursus ini?

- "Aku pakai aplikasi setiap hari, tapi aku **sama sekali tidak tahu bagaimana semuanya benar-benar bekerja**."
- "Aku bekerja *bersama* para insinyur (manajer, desainer, pendiri startup, pengacara, dokter…) dan aku
  capek **cuma mengangguk-angguk di rapat**."
- "Aku ingin **pindah karier ke dunia teknologi**, tapi setiap kursus menganggap aku sudah tahu hal-hal yang tidak kutahu."
- "Aku sekadar **penasaran** bagaimana dunia digital di sekitarku dibangun."

**Tidak ada prasyarat apa pun** — kami menjelaskan segalanya, termasuk apa itu "berkas (file)" sebenarnya.

## 🎓 Apa yang akan kamu pelajari

- Bagaimana komputer, perangkat lunak, internet, basis data (database), cloud, dan AI benar-benar bekerja.
- Cara membaca dan menulis program sederhana dengan Python.
- Cara memakai Git dan GitHub layaknya profesional (kamu akan menerbitkan situs web live milikmu sendiri).
- Bagaimana tim perangkat lunak sungguhan merencanakan, membangun, menguji, dan merilis produk.
- Cara berbicara dengan insinyur sebagai orang yang setara — API, server, bug, deploy, dan sprint, semua dibongkar artinya.

**Yang tidak akan kami bahas:** matematika tingkat lanjut, alat enterprise tertentu, atau
gembar-gembor kosong. Setiap analogi juga memberi tahu di mana batas berlakunya, dan modul AI
jujur soal keterbatasannya.

## 📝 Kuis & Lab

- **Kuis** ada di dalam setiap folder modul (`quiz.md`) — jawabannya disertakan di balik
  lipatan "Show answer", dan setiap kuis diakhiri dengan tantangan *"jelaskan ke seorang teman"*.
- **Lab** (`lab.md`) bersifat praktik langsung dan **tidak butuh instalasi sama sekali** — hanya
  alat gratis berbasis peramban, ramah ponsel. Mengerjakan lab-lah yang membuat pelajaran benar-benar melekat.
