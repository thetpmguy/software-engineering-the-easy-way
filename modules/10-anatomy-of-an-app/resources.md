# 📚 Resources 10 — Go Deeper on Frontend, Backend & APIs

All free. This module opens a door many learners want to walk through — actually
*building* web pages. The first two resources are the standard ways in.

---

## 🌐 Learn by reading (the standard reference)

- **[MDN Web Docs — beginner guides](https://developer.mozilla.org)** — MDN (from
  Mozilla, the Firefox people) is *the* reference the world's web developers use daily,
  and its "Learn web development" section is a genuinely gentle on-ramp: what HTML, CSS,
  and JavaScript are, with live examples you can edit in the page. **Best for:** anyone
  who wants trustworthy, ad-free depth on the skeleton/clothing/muscles trio. When a
  web question matters, the professional habit is: check MDN first.

## 🛠️ Learn by building

- **[freeCodeCamp](https://www.freecodecamp.org)** — a famous nonprofit offering
  complete, free, in-browser courses; its Responsive Web Design certification teaches
  HTML and CSS by having you build real pages from the first minute, no installation.
  **Best for:** learners who finished this module thinking "I want to build the dining
  room myself." Thousands of people have gone from zero to employed through
  freeCodeCamp; it pairs perfectly with everything this course has taught you.

- **[Codecademy — free HTML course](https://www.codecademy.com)** — interactive
  type-in-the-browser lessons with instant feedback; the free tier's HTML basics course
  is a pleasant, structured first taste. **Best for:** learners who like guided,
  bite-sized steps with a progress bar. (Watch the upsell: the free tier is real, but
  the site will offer you paid "Pro" constantly. Everything you need at this stage is
  free — here or at freeCodeCamp.)

## 🎬 Watch

- **[Crash Course Computer Science — the World Wide Web episode (and neighbors)](https://www.youtube.com/playlist?list=PL8dPuuaLjXtNlUrzyH5r6jN9ulIgZBpdo)** —
  episode 30 ("The World Wide Web") shows how pages, links, and browsers actually work,
  and the surrounding episodes (the internet, networks) rhyme beautifully with our
  Module 03 and this one. **Best for:** visual learners wanting the frontend/backend
  story connected to the internet plumbing underneath it.

## 💡 One tip

The most valuable free resource from this module is already installed: **DevTools**.
The lab habit — right-click → Inspect, and a glance at the Network tab — turns every
website you ever visit into an anatomy lesson. Dissect freely; refresh heals all.
